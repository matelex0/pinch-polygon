# 🖐️ Pinch Polygon - Webcam AI Tool

A simple and fun **AI experiment** that lets you control geometric shapes with your hands!

## 🚀 How it works

1.  Show your hand to the webcam.
2.  **Spread your fingers** (Thumb & Index) 👉 **Increase** polygon sides.
3.  **Pinch your fingers** (Thumb & Index) 🤏 **Decrease** polygon sides.
4.  Press `Q` to exit.

It's that simple!

## 📦 Installation

1.  **Clone the repo**
    ```bash
    git clone https://github.com/alex1dev0/pinch-polygon.git
    cd pinch-polygon
    ```

2.  **Install dependencies**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run it!**
    ```bash
    python tool.py
    ```
